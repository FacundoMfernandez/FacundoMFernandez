🔧 **[ens-hardening-ubuntu](https://github.com/FacundoMfernandez/ens-hardening-ubuntu)** — Hardening de Ubuntu 22.04 según CIS Benchmarks + ENS Categoría Alta. Auditoría pre/post con Lynis.
